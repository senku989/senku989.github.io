📌 نظام الري الذكي باستخدام Arduino

📌 Smart Irrigation System Using Arduino

🔹 مقدمة

يهدف هذا المشروع إلى تطوير نظام ري ذكي يعتمد على مستشعرات رطوبة التربة ودرجة حرارة الجو لضمان ري النباتات فقط عند الحاجة.

🔹 Introduction

This project aims to develop a smart irrigation system that uses soil moisture and air temperature sensors to ensure plants are watered only when needed.

🔹 مكونات المشروع

🔹 Project Components

المكون الوظيفة Component Function
Arduino Uno وحدة التحكم الرئيسية Arduino Uno Main control unit
مستشعر رطوبة التربة ×2 لقياس مستوى الرطوبة لكل شجرة Soil Moisture Sensor ×2 Measures soil moisture level for each tree
مستشعر درجة حرارة الجو لقياس حرارة الهواء لتجنب الري في الحرارة العالية Air Temperature Sensor Measures air temperature to avoid irrigation in high heat
محرك سيرفو فتح وإغلاق صمام الماء Servo Motor Opens and closes the water valve
خزان ماء مصدر المياه للنباتات Water Tank Water source for plants

🔹 نظام الري لكل شجرة

🔹 Irrigation System for Each Tree

الشجرة الحد الأدنى لرطوبة التربة قبل الري الحد الأقصى لدرجة الحرارة لتشغيل الري Tree Minimum Soil Moisture Before Irrigation Maximum Temperature to Allow Irrigation
التين أقل من 40% أقل من 40°C Fig Tree Less than 40% Less than 40°C
النخيل الصغير أقل من 25% أقل من 40°C Small Palm Tree Less than 25% Less than 40°C

🔹 طريقة عمل النظام

يقوم النظام بقراءة بيانات المستشعرات لكل شجرة على حدة، ثم يقرر تشغيل الري إذا كانت التربة جافة ودرجة الحرارة مناسبة. عند تحقيق المستوى المطلوب من الرطوبة، يتم إيقاف الري تلقائيًا.

🔹 How the System Works

The system reads sensor data for each tree separately and decides to start irrigation if the soil is dry and the temperature is appropriate. When the desired moisture level is reached, irrigation stops automatically.
